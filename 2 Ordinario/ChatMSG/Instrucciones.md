# Activdad:

Crear una aplicación de mensajería (chat).
Utilizar React para el frontend y express para el Backend.

## Primera etapa:
-> No usar Redis
-> Sin persistencia (todo en memoria).
-> Sin cookies httpOnly

## Segunda etapa:
-> Incorporar redis remoto (remoto <---> sus/pub).
-> Agregar cookies httpOnly
-> Sin persistencia
-> Al menos dos servidores
